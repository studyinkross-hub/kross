⚙️ Hướng Dẫn Cài Đặt
Chỉ cần cài đặt một lần. Sau đó bài tập sẽ tự động lưu vào Google Drive.
1
Tạo thư mục trên Google Drive Tạo thư mục tên KROSS_숙제 trên Drive, sau đó copy ID từ URL.
Ví dụ: https://drive.google.com/drive/folders/1AbCdEfGh…
2
Tạo Google Sheets Tạo một bảng tính mới và copy ID từ URL.
Ví dụ: https://docs.google.com/spreadsheets/d/1XxYyZz…/edit
3
Dán code vào Apps Script Vào script.google.com → Tạo dự án mới → Dán toàn bộ nội dung file kross_gas.js
Điền Folder ID, Sheet ID và mật khẩu vào phần CONFIG ở đầu file.
4
Triển khai dưới dạng Web App Nhấn Triển khai → Triển khai mới
Loại: Ứng dụng web / Chạy với tư cách: Tôi / Quyền truy cập: Mọi người
→ Copy URL Web App sau khi triển khai.
5
Dán URL vào file homework.html Tìm dòng const GAS_URL = ''; và dán URL vào trong cặp nháy đơn.
Lưu file và upload lên GitHub → Hoàn tất! 🎉
